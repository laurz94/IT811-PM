Project Overview

Project aim: The goal of this project is to not only improve on the current KU Bookstore functionality but create additional relevant content. Functionally relevancy and student aid are key to this goal, rather than pure profit center model. It is supposed to unite students around the KU Bookstore as a foundation, rather than a burden. Streamlined Payment Process Merchandise upgrade & Features Student Services.

Objectives

  • Create an easy payment process
  • Allow for multiple payment methods
  • Create simulated merchandise previews
  • Sell tickets university games and events
  • Automatically match the price of competing sites
  • Allow searching for products based on your major
  • Create a discount hub to easily find items that are offered at a discounted rate
  • Create a community hub to allow students to a place to connect and interact with each other
  • Allow for in-text book search so students can find books related to specific topic
  • Allow for scheduling outdoor pick-up/drop-off of materials
  • Allow the ability to create an account online
  • Allow the ability for students to share feedback about the bookstore

Key Assumptions

  • Interface will be a web browser
  • Seamless integration with the KU athletics API
  • Social media department will run a media blitz to advertise the new website
  • Social media department will post campus events on the website
  • IT department will provide integration into student database or account creation
  • APIs available for price matching on Amazon and other competitor websites
  • Textbook publishers will provide APIs for searching the contents of the textbooks
  • IT department will establish a mailbox for bookstore feedback
